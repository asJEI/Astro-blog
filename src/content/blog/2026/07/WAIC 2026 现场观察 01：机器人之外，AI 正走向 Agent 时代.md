今年是我第一次参加 WAIC。走进上海世博展览馆之后，最直观的感受就是：**怎么到处都是机器人。**

无论是展台展示，还是企业宣传，机器人几乎成为了这次大会最具视觉冲击力的存在。双足机器人、机械臂、机器牛，还有机器豆包……

![WAIC 现场：带有动漫风格头部的四足机器人](/images/blog/07/image-31.png)

各种各样的机器人，在初次见到之后确实很震撼。但逛完一天之后，我反而觉得：**机器人可能并不是 AI 未来最核心的方向。**

## 机器人：AI 进入现实世界的载体，而不是智能本身

不可否认，机器人技术正在快速发展。但从现场观察来看，目前很多机器人产品，本质上还是**视觉模块 + 机械臂**。

它们能够识别环境，也能够完成动作，但距离真正意义上的智能体机器人，还有一定距离。

例如，机械臂可以完成抓取，但它并不知道：为什么要抓取？抓取之后应该做什么？遇到异常情况应该如何处理？它更多是在执行任务，而不是理解目标。

双足机器人也是类似。它们拥有非常强的视觉冲击力，但距离真正进入生产环境，还需要解决：

- 成本问题
- 能源问题
- 灵活性问题
- 任务泛化问题

因此，我认为机器人更像是：

> AI 进入现实世界的一种载体。

真正决定未来 AI 能力上限的，并不是机器是否拥有人的身体，而是 AI 是否拥有理解目标、规划任务、调用工具并完成工作的能力。而这正是 Agent 关注的问题。

## 从大模型到 Agent：AI 正在进入新的阶段

在 WAIC 主论坛开幕式上，阶跃董事长、千里科技董事长印奇分享了他对于 AI 发展路径的理解。回顾 AI 过去 70 年的发展，可以发现一个明显趋势：

> AI 一直在尝试让机器获得越来越接近人类的能力。

从最初的规则推理，到数据学习，再到如今的大模型和 Agent，AI 与现实世界之间的连接方式也在不断变化。

### 符号主义阶段

早期 AI 采用的是符号主义路线。这一阶段的核心思想是：

> 如果人类能够描述世界，那么就可以把这些知识直接写入机器。

因此研究人员尝试通过大量人工编写的规则，让计算机模拟人类推理过程。例如：「如果 A，那么 B。」「如果满足条件 X，则执行 Y。」

这种方法在一些明确的问题中非常有效，例如数学证明、专家系统等。但问题也很明显：现实世界并不是由简单规则组成的。人的一句话、一个动作、一个环境变化，都可能包含大量隐藏信息。人类可以根据经验处理模糊情况，但机器很难提前拥有所有规则。

因此，符号主义最大的限制在于：

> 人类无法把现实世界完整地描述出来。

### 机器学习阶段

随着数据规模增长和计算能力提升，AI 开始走向机器学习阶段。这一阶段的重要变化是：机器不再需要人类告诉它所有规则；相反，人类提供大量数据，让机器自己寻找其中的规律。

例如：通过大量图片，机器学习如何识别猫和狗；通过历史数据，机器预测未来趋势；通过用户行为，机器推荐可能感兴趣的内容。AI 开始拥有：

- 分类能力
- 预测能力
- 模式识别能力

从某种意义上来说，这是 AI 第一次真正开始接近人类学习方式：

> 不依靠固定规则，而是通过经验形成能力。

### 深度学习与大模型阶段

深度学习进一步推动了 AI 能力提升。尤其是大语言模型出现之后，AI 开始拥有前所未有的能力：

- 理解自然语言
- 生成文本内容
- 总结复杂信息
- 进行一定程度的推理

语言成为 AI 连接人类知识世界的重要入口。过去，人类需要通过软件界面操作计算机；而大模型让人可以直接通过自然语言表达需求。

例如，过去是「打开搜索引擎 → 输入关键词 → 筛选信息」；现在可以直接告诉 AI：「帮我整理这个领域的发展趋势。」这是一种交互方式上的变化。

但大模型仍然存在一个限制：它更像一个非常聪明的顾问。它能够分析问题、提供建议，但最终执行任务的人仍然是人。

它可以告诉你「应该如何开发一个网站」，但它不会自动：

- 创建项目
- 编写代码
- 部署服务
- 检查运行结果

因此，大模型解决的是：

> AI 如何理解和生成信息。

而 Agent 进一步解决的是：

> AI 如何利用信息改变现实。

## Agent 时代的三个变化

如果说大模型让 AI 获得了理解世界的能力，那么 Agent 时代真正改变的，是 AI 与现实世界交互的方式。未来的变化可能不仅仅体现在模型能力提升，而会进一步影响：

- 操作系统
- 智能终端
- 网络结构

### 1. Agent OS：围绕智能体构建的操作系统

每一次计算平台的变化，都会催生新的操作系统。

- PC 时代：Windows、macOS
- 移动时代：Android、iOS

操作系统的发展，本质上是在解决一个问题：

> 如何管理人与计算设备之间的关系。

那么进入 Agent 时代之后，一个新的问题出现：

> 如果未来每个人都拥有多个 Agent，谁来管理这些智能体？

因此，我认为未来可能会出现类似 Agent OS 的新型操作系统。它管理的不再只是文件、应用、硬件资源，而可能包括：

- Agent 身份
- 权限边界
- 长期记忆
- 工具调用
- Agent 之间的协作关系

过去：

> 操作系统帮助人管理计算机。

未来：

> 操作系统可能帮助人管理智能体。

在这个过程中，大模型更像是提供智能的「大脑」，而 Agent OS 则负责让这个智能真正连接现实世界。例如：一个 Agent 想访问用户邮箱，它需要权限；想修改文件，它需要授权；多个 Agent 想协作，它们需要通信协议。这些问题都不是单纯提升模型能力能够解决的。

因此，未来 Agent 时代的重要竞争，不仅是模型能力竞争，也是基础设施竞争。当然，这其中仍然存在大量挑战：

- 数据隐私
- 权限控制
- 安全问题
- 责任归属

这些都会成为 Agent 时代必须建立的新规则。

### 2. 新的智能终端：AI 不会只存在于手机

计算平台的发展，往往伴随着终端形态的变化。

过去：

> 人操作设备，设备提供信息。

例如：打开手机 App，主动搜索内容。

未来：

> Agent 感知环境，并主动帮助人完成任务。

用户不一定需要打开应用，而是直接表达目标。例如「帮我安排一次旅行」，Agent 可以主动：

- 查询航班
- 分析天气
- 调整行程
- 提醒用户

因此，未来 AI 终端可能不会只有手机一种形态。它可能存在于：

- AI 眼镜
- 智能耳机
- 汽车
- 可穿戴设备
- 机器人

在 WAIC 现场，我体验了千问 AI 眼镜。这让我意识到：未来终端可能不存在唯一答案——有人习惯手机，有人依赖眼镜，有人更适合手表或者其他设备。

真正重要的不是终端本身，而是：

> 每一种终端背后，都可能连接着一个长期存在的个人 Agent。

未来的设备可能从「人使用工具」，逐渐变成「Agent 陪伴并服务于人」。

### 3. Agent 网络：未来可能出现 Agent 之间的互联网

互联网改变世界的重要原因，是它连接了人与信息。

过去：

> 人 ↔ 信息 ↔ 服务

人通过互联网获取信息，并使用各种数字服务。而未来，随着 Agent 数量增加，连接关系可能进一步变化：

> Agent ↔ Agent

不同领域的 Agent 可以互相协作。例如：产品 Agent 发现用户需求变化，通知研发 Agent；研发 Agent 分析问题并修改代码；测试 Agent 自动验证结果；运营 Agent 根据反馈调整策略——最终形成一个由多个智能体共同完成任务的网络。

从这个角度来看，未来的互联网可能不仅连接人和信息，也会连接大量自主运行的智能体。当然，这也带来了新的问题：

- 一个 Agent 是否可信？
- 一个 Agent 的行为由谁负责？
- 不同 Agent 之间如何建立信任？

这些问题可能需要新的身份体系、验证机制和通信协议。

某种程度上，这和互联网早期的发展非常相似：最开始，人们只是连接计算机；后来，人们连接信息；未来，人们可能连接智能体。**Agent 网络可能成为下一代数字世界的重要基础设施。**

## 我的判断：AI 未来可能首先成为基础设施，而不是机器人

这次 WAIC 给我最大的感受是：AI 的未来可能不会首先表现为一个更加像人的机器人，而更可能表现为一种无处不在的智能基础设施。

机器人当然重要，它解决的是 AI 如何进入物理世界的问题。但从目前的发展阶段来看，真正决定 AI 能否改变生产方式的，并不只是机器是否拥有身体，而是 AI 是否能够真正融入人的工作流程。

未来，AI 可能逐渐进入：

- 软件
- 操作系统
- 智能终端
- 企业流程

它可能存在于手机里、眼镜里、汽车里，也可能存在于企业内部系统中，甚至存在于无数 Agent 组成的协作网络里。

就像过去互联网并没有改变计算机本身，而是改变了信息连接方式一样，未来 AI 可能也不会首先通过一个「像人一样的机器」改变世界，而是通过智能能力的普及改变整个社会运行方式。

机器人只是 AI 连接现实世界的一种形式。而 Agent，则可能成为 AI 从「理解世界」走向「参与世界」的关键路径。

## 文末

从这次 WAIC 现场来看，AI 产业正在经历一次方向变化。

过去几年，AI 竞争的核心更多集中在：

- 模型规模
- 参数数量
- 算力资源

这些决定了 AI 能力的上限。但未来，竞争可能逐渐转向：

- Agent 能力
- 工具调用能力
- 任务执行能力
- 人机协作能力

因为真正影响生产力的，并不仅仅是一个模型能够回答多少问题，而是它能否真正帮助人完成任务。

AI 的未来，可能并不是创造一个完全替代人类的机器，而是创造一种新的生产方式：

> 人 + Agent。

未来的人类，也许不会因为 AI 而失去价值，而是通过 AI 获得一种新的协作伙伴。真正重要的能力，可能不再只是单纯执行任务，而是提出目标、组织资源、协调智能。

这可能才是 AI 下一阶段真正改变世界的地方。
